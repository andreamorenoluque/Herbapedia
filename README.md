# Herbapedia 

<img width="328" height="631" alt="image" src="https://github.com/user-attachments/assets/23f5fb9e-678c-4660-9fea-6c43a7b424e1" />

**Herbapedia** es una aplicación móvil que ofrece una biblioteca digital de plantas, árboles frutales y hortalizas que los usuarios pueden consultar y enriquecer con sus propias aportaciones. Esta herramienta tiene como objetivo facilitar el acceso a una base de datos de conocimiento en constante crecimiento y mantener un espacio dinámico e intuitivo donde los aficionados a la agricultura y la botánica puedan intercambiar información útil y aprender unos de otros. La aplicación se desarrollará utilizando el framework **.NET MAUI**, aprovechando sus capacidades multiplataforma, para que sea compatible con Android, iOS y ordenadores de escritorio. Además, la base de datos estará gestionada por **PostgreSQL**, elegida por su robustez y capacidad de gestión de grandes volúmenes de datos. Para la interfaz de usuario, utilizaremos tecnologías de diseño modernas y **C#**, un lenguaje de programación, para garantizar una experiencia de usuario fluida y un rendimiento óptimo. El sistema de chat y foro, así como los comentarios para la interacción del usuario, una base de datos que almacena imágenes e información útil sobre cada planta, y un panel de gestión para revisar y verificar las contribuciones, asegurando la calidad de los datos almacenados, son, en resumen, las características más destacadas de esta aplicación.
